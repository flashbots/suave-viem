# suave-web-demo

Simple Vanilla JS web app that sends a mev-share bid to suave.

## setup

Go to the [suave example](../suave/) and run `./deployContracts.sh`:

```sh
cd ../suave
./deployContracts.sh
cd -
```

Next, install the project's dependencies.

```sh
bun install
```

Finally, run the webserver:

```sh
bun dev
```
